Inspetor de elementos

(Voltar)

O inspetor de elementos é a ferramenta do desenvolvedor, que é utilizada e fornecida pelo navegador. No navegador Google Chrome, para abrir o inspetor de elementos, pressionamos a tecla F12. Ao pressionar F12 um painel aparecerá no lado direito da tela, sendo possível configurar para que apareça na parte inferior ao invés do lado direito. Outra forma de abrir o inspetor de elementos é clicando nos três pontinhos no canto superior direito do navegador, mais ferramenta, ferramenta do desenvolvedor. Na ferramenta existem recursos que nos permitem visualizar como a página foi construída, ou seja, a sua estrutura HTML.

Estrutura básica do HTML

(Voltar)

Para começar, definimos o nome do nosso arquivo que, por padrão, chama-se index.html. Esse é o arquivo onde construiremos a página principal do nosso site.

O nosso arquivo HTML divide-se basicamente em três partes: Nas tags html, head e body. A tag html é onde ficarão todas as outras tags que usaremos para construir a nossa página. Na tag head é onde ficam as coisas que serão carregadas antes de serem exibidas para o usuário. Na tag body ficará o corpo da nossa página, ou seja, o texto da nossa página.

Tags

(Voltar)

Na aula sobre tags nos foram apresentadas alguma tags básicas. Essas tags e o que cada uma delas faz está listado adiante.

  • Tag i;
  • Tag u;
  • Tag strong;
  • Tag img;
  • Tag input.

A tag i serve basicamente para colocar o texto em itálico.

A tag u serve para sublinhar o texto. O "u" que dá nome a tag vem de underline.

A tag strong deixa o texto em negrito.

A tag img serve para adicionarmos uma imagem à nossa página, porém, para isso é necessário o uso de um atributo, que será explicado no próximo tópico.

A tag input é um pouco diferente das outras, ela por si só já é um campo, ou seja, ela não precisa de uma tag de fechamento, assim como a tag img também não precisa.

Atributos das tags

(Voltar)

Atributos são propriedades que uma tag pode ter. Existem atributos "globais", ou seja, todas as tags vão ter, e existem atributos que são mais específicos de cada tag. Abaixo temos alguns atributos:

id: O atributo id é um identificador desse elemento. É muito utilizado para trabalhar com JavaScript.

style: O atributo style serve para colocar comandos CSS direto no elemento.

class: Tudo que tiver com o que foi definido dentro do atributo class, por exemplo, a palavra título-principal, tudo que tiver título-principal, ficará com um formato pré-definido no CSS.

Agora segue alguns atributos mais específicos de algumas tags:

type: O type é um atributo específico da tag input, que pode receber texto, número, cor, dentre outros tipos. Por exemplo, ao criarmos uma tag input com o atributo type recebendo number aparecerá, na página do navegador, uma caixa de digitação, que aceita apenas números.

src: O src é utilizado com a tag img, e indica o caminho onde a imagem está, seja um site na web ou um caminho no computador.

Listas ordenadas e não ordenadas

(Voltar)

Na aula sobre listas, vimos dois tipos de listas: as listas ordenadas e as listas não ordenadas.

As listas ordenadas são aquelas que seguem uma sequência numérica, iniciando do número 1.

As listas não ordenadas não seguem uma sequência, são apenas itens dispostos em uma lista.

Para as listas ordenadas utilizamos a tag "ol" e para as listas não ordenadas utilizamos a tag "ul". Além dessas duas tags, utilizamos também a tag "li", que serve para adicionar os itens da lista, já que apenas criar uma lista não é o suficiente para sairmos digitando itens.

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Exemplo de lista não ordenada:

  • Item 1
  • Item 2
  • Item 3
