Propriedade word-wrap

Define, caso uma palavra seja muito grande para caber em uma linha, se ela será quebrada em qualquer caractere para que uma parte dela fique na linha e o restante na próxima, ou se não, toda a palavra deve ficar na mesma linha e ultrapassar os limites que definimos para os elementos.

word-wrap: normal;
